# 콘텐츠 관리 가이드 — 광주새서광교회

## 설교 추가하기

1. GitHub 저장소 → Issues → "New Issue" 클릭
2. "새 설교 추가" 템플릿 선택
3. YouTube URL 붙여넣기
4. "Submit new issue" 클릭
5. 자동으로 웹사이트에 반영됩니다 (약 2-3분)

## 주보 추가하기

1. GitHub 저장소 → Issues → "New Issue" 클릭
2. "새 주보 추가" 템플릿 선택
3. 주보 내용 작성
4. "Submit new issue" 클릭
5. 자동으로 웹사이트에 반영됩니다

## 주의사항

- 설교 영상은 YouTube에 먼저 업로드한 후 URL을 복사하세요
- 주보는 매주 일요일 예배 전에 추가하시면 좋아요
- 2주 이상 주보가 없으면 "카카오톡에서 만나보세요" 배너가 자동 표시돼요
